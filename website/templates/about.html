So, here’s why you should learn flask ?

  • It is Light Weight
    • Flask is a micro framework. Micro meaning Flask aims to keep the core simple but extensible, there is not much required to get up and running with flask
  • Extensible and flexible
    • Extensible ? Yes so flask has what are called extensions they are python modules that add functionality and support to our flask application. Flask implements bare-minimum core functionality and lets you decide how you would like to extend it
  • Easy to learn
    • It's very beginner friendly and one can learn flask within few days
In my opinion, flask is probably the best web framework out there and you should consider learning and using it.
